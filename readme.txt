Define specific tabs for the description of your products2. Customize your product descriptions in line with your product range= Custom Tab Free Features: =* For every tab you get to give a title to your tab as well as the space where you can add the description that is line with the title that you have given to that tab.* Any additional tabs that you create with the Custom Tabs plugin will be applicable to all the products on your website.* The custom tabs plugin gives you the freedom to create as many tabs as you see fit for your product range.= Premium Features: =* Global Tab Option allows you to have a common tab on all of the product pages, irrespective of their product categories.* Category Tab Option allows you to show a tab that is common only to same-category products. Tab specifications could only vary from category to category.* Product-Specific Tab Option lets you have a tab that is unique to a particular product page.* Tab Placement Option lets you sequence your tabs as per your priority.* You could  Sort, Re-order, Rename or Enable/Disable the default tabs as per your requirement.* You can also add shortcode in the custom tabs.[LIVE DEMO](http://customtab.phoeniixxdemo.com/?product=demo-product) | [PREMIUM VERSION LINK](https://www.phoeniixx.com/product/woocommerce-custom-tabs-plugin/)= Premium WordPress Themes =Here is our premium wordpress theme* <a href="https://www.phoeniixx.com/product/foody/">Foody Themes</a>* <a href="https://www.phoeniixx.com/product/jstore-theme/">Jstore Themes</a>* <a href="https://www.phoeniixx.com/product/eezy-store/">Eezy Themes</a>* <a href="https://www.phoeniixx.com/product/craze/">Craze Themes</a><blockquote><h4>Our Website</h4><ul><li>  <a href="https://www.phoeniixx.com/">https://www.phoeniixx.com/</a></li></ul><h4>Support Email</h4><ul><li>  <a href="mailto:support@phoeniixx.com">support@phoeniixx.com</a></li></ul><h4>NOTE</h4><ul><li>  Our plugin doesn't support the multisite network.</ul><p></p></blockquote>== Installation ==1. Install [WordPress](https://wordpress.org/)2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.3. For the plugin to work as it should, WooCommerce plugin has to be installed and enabled.== Frequently Asked Questions ==** Coming soon== Screenshots ==1.
